# Section 1: Dimensionality Reduction

## Overview
This section covers dimensionality reduction techniques including PCA and SVD analysis.

## Code Files
- `pca_mean_filling.py` - PCA with mean filling for missing values
- `pca_mle.py` - PCA with MLE estimation
- `svd_analysis.py` - SVD analysis implementation
- `utils.py` - Utility functions

## Directories
- `data/` - Input datasets
- `results/` - Output results
- `plots/` - Generated visualizations
- `tables/` - Generated tables
