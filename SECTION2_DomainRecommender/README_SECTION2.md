# Section 2: Domain Recommender

## Overview
This section implements a domain-specific recommender system using content-based, collaborative, and hybrid approaches.

## Code Files
- `data_preprocessing.py` - Data preprocessing and cleaning
- `content_based.py` - Content-based filtering implementation
- `collaborative.py` - Collaborative filtering implementation
- `hybrid.py` - Hybrid recommender combining approaches
- `main.py` - Main execution script

## Directories
- `data/` - Input datasets
- `results/` - Output results and evaluation metrics
